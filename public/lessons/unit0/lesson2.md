# More Basic Korean Letters

Source: [How to Study Korean - Unit 0, Lesson 2](https://www.howtostudykorean.com/unit0/0-lesson-2/)

In the previous lesson, we studied the most basic letters in the Korean alphabet. There are still some more letters that you will need to wrap your head around before we go any further.

## The Letter ㅇ

The first new letter that you will learn is: **ㅇ**

When **ㅇ** is placed at the beginning of a syllable (position 1), it is **silent**, and the first sound that is made is the vowel:

| Syllable | Romanization |
|----------|-------------|
| 안 | an |
| 운 | un |
| 온 | on |
| 업 | eob |

When **ㅇ** is placed at the end of a syllable (position 3), it has the sound of **'ng'** as in "walki**ng**":

| Syllable | Romanization |
|----------|-------------|
| 강 | kang |
| 방 | bang |
| 깅 | king |
| 공 | kong |

The letter **ㅇ** can be placed as both position 1 and 3: **앙** = ang

## New Consonant Sets

### ㄱ Set

| Korean | Romanization | Description |
|--------|-------------|-------------|
| ㄱ | k | Original letter |
| ㄲ | kk | Double consonant, more forced |
| ㅋ | k | Aspirated, closer to English 'k' |

### ㅂ Set

| Korean | Romanization | Description |
|--------|-------------|-------------|
| ㅂ | b | Original letter |
| ㅃ | bb | Double consonant, more forced |
| ㅍ | p | Aspirated, closer to English 'p' |

### ㅈ Set

| Korean | Romanization | Description |
|--------|-------------|-------------|
| ㅈ | j | Original letter |
| ㅉ | jj | Double consonant, more forced |
| ㅊ | ch | Aspirated, closer to English 'ch' |

### ㄷ Set

| Korean | Romanization | Description |
|--------|-------------|-------------|
| ㄷ | d | Original letter |
| ㄸ | dd | Double consonant, more forced |
| ㅌ | t | Aspirated, closer to English 't' |

### ㅅ Set

| Korean | Romanization | Description |
|--------|-------------|-------------|
| ㅅ | s | Original letter |
| ㅆ | ss | Double consonant, more forced |

### Example Syllables

| Combination | Result | Romanization |
|-------------|--------|--------------|
| ㄸ + ㅏ + ㅇ | 땅 | ddang |
| ㅌ + ㅗ + ㅇ | 통 | tong |
