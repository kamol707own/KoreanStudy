# Korean Diphthongs: Complex Vowels

Source: [How to Study Korean - Unit 0, Lesson 3](https://www.howtostudykorean.com/unit0/unit-0-lesson-3/)

In this final lesson of Unit 0, you will learn the remaining vowels that you will need to be able to read Korean. Most of these letters are called "diphthongs" — which is just a fancy way of saying that there are two vowels in one.

## Review of Basic Vowels

| Korean | Romanization |
|--------|-------------|
| ㅏ | a |
| ㅓ | eo |
| ㅜ | u |
| ㅗ | o |

## Y-Added Vowels

By adding one extra line to each vowel, a 'Y' sound gets added:

| Korean | Romanization | Example |
|--------|-------------|---------|
| ㅑ | ya | 약 = yak |
| ㅕ | yeo | 역 = yeok |
| ㅠ | yu | 육 = yuk |
| ㅛ | yo | 욕 = yok |

## ㅐ and ㅔ

| Korean | Romanization | Notes |
|--------|-------------|-------|
| ㅐ | ae | Sounds like "eigh" in "weigh" |
| ㅔ | e | Sounds identical to ㅐ |

> Both ㅐ and ㅔ sound exactly the same to modern Korean ears. For all practical purposes, especially for a foreigner learning Korean, the sounds are identical.

**Examples:**
- 배 = bae (sounds like bay)
- 베 = be (also sounds like bay)

**Audio examples:**
- ㅐ with consonants: 애, 배, 재, 대, 개, 새, 매, 내, 해, 래
- ㅔ with consonants: 에, 베, 제, 데, 게, 세, 메, 네, 헤, 레

> (Yes, it's exactly the same — ㅐ and ㅔ sound exactly the same)

## Combined Vowels

| Combination | Result | Romanization | Example |
|-------------|--------|-------------|---------|
| ㅜ + ㅣ | ㅟ | wi | 쥐 |
| ㅜ + ㅓ | ㅝ | wo | 원 |
| ㅗ + ㅣ | ㅚ | oe (sounds like "way") | 외국 |
| ㅗ + ㅏ | ㅘ | wa | 완 |
| ㅡ + ㅣ | ㅢ | ui | 의 |
| ㅕ + ㅣ | ㅖ | ye | 예 |

**Combined vowel syllables:**

| Korean | Romanization |
|--------|-------------|
| 위, 뷔, 쥐, 뒤, 귀, 쉬, 뉘 | wi |
| 워, 붜, 줘, 둬, 궈, 숴, 눠 | wo |
| 외, 뵈, 죄, 되, 괴, 쇠, 뇌 | oe |
| 와, 봐, 좌, 돠, 과, 솨, 놔 | wa |
| 의, 븨, 즤, 듸, 긔, 싀, 늬 | ui |
| 예, 계, 혜 | ye |

> Notice that when one of these "double vowels" is used, the first consonant of the syllable is added on top of the horizontally aligned letter and to the left of the vertically aligned letter at the same time.

## Less Common Vowels

| Korean | Romanization | Notes |

## Four-Letter Syllables

It is also possible for a syllable to have four letters: one vowel and three consonants. The third consonant gets added beside the second consonant.

**Common examples:**

| Korean | Romanization | Meaning |
|--------|-------------|---------|
| 닭 | dak | chicken |
| 앉다 | antda | to sit |
| 읽다 | ikda | to read |
| 없다 | eopda | to not have |
| 긁다 | geulda | to scratch |

> Pronouncing syllables with this fourth letter is slightly more complicated. The pronunciation depends on if the upcoming syllable starts with a vowel or a consonant.

| 소파 | sofa |
| 텔레비전 | television |

| 게임 | game |
| 쇼핑 | shopping |
| 오렌지 | orange |

| 택시 | taxi |
| 피자 | pizza |
| 햄버거 | hamburger |
| 샤워 | shower |

> Korean sound structure is vastly different from English, so sometimes these Korean words sound weird compared to their English counterpart.
