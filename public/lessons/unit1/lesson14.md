# Lesson 14: Korean Passive Verbs

Source: [How to Study Korean - Unit 1, Lesson 14](https://www.howtostudykorean.com/unit1/unit-1-lessons-9-16/lesson-14-korean-passive-verbs/)

## Active vs Passive

| Active | Passive |
|--------|---------|
| I opened the door | The door was opened |
| I turned on the computer | The computer was turned on |
| I respect my friend | My friend is respected |

> Passive verbs cannot act on an object (~을/를 is never used with them)

## Making Passive Verbs

### 하다 → 되다 (most common)

| Active | Passive |
|--------|---------|
| 포함하다 (to include) | 포함되다 (to be included) |
| 제공하다 (to provide) | 제공되다 (to be provided) |
| 대체하다 (to replace) | 대체되다 (to be replaced) |

**Examples:**
- 점심이 학교에 준비되었어요 = The lunch was prepared by the school
- 기계는 대체되었어요 = The machine was replaced

**By whom/what particles:**
- 에게 = by a person
- 에 = by a non-person
- ~(으)로 = by tool/method

### 하다 → 받다 (for people-related actions)

| Active | Passive |
|--------|---------|
| 존경하다 (to respect) | 존경받다 (to be respected) |
| 감동하다 (to impress) | 감동받다 (to be impressed) |

### Non-하다 Verbs (separate passive words)

| Active | Passive |
|--------|---------|
| 켜다 (to turn on) | 켜지다 (to be turned on) |
| 끄다 (to turn off) | 꺼지다 (to be turned off) |
| 닫다 (to close) | 닫히다 (to be closed) |
| 열다 (to open) | 열리다 (to be opened) |
| 잠그다 (to lock) | 잠기다 (to be locked) |

### State of Being: ~아/어 있다

For things that are "in the state of" something:

| Sentence | Meaning |
|----------|---------|
| 컴퓨터가 켜져 있어요 | The computer is (in the state of) on |
| TV가 꺼져 있어요 | The TV is off |
| 문이 잠겨 있어요 | The door is locked |
| 문이 닫혀 있어요 | The door is closed |

### 나다/내다 Verbs

| Active | Passive |
|--------|---------|
| 끝내다 (to finish) | 끝나다 (to be finished) |
| 고장내다 (to break) | 고장나다 (to be broken) |

**Common usage:**
- 숙제는 끝났어요 = My homework is finished
- 컴퓨터는 고장 났어요 = The computer is broken
- 기억이 나다 = to remember (a memory comes up)
- 생각이 나다 = to have an idea
- 냄새가 나다 = to be smelly
- 화가 나다 = to be angry

### Adding ~아/어지다

| Active | Passive |
|--------|---------|
| 주다 (to give) | 주어지다 (to be given) |
| 짓다 (to build) | 지어지다 (to be built) |

> **Key takeaway:** Active sentences are almost always more natural in Korean than passive sentences.
