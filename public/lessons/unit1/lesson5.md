## How to Say "I" or "me"

| Korean | Usage | When ~가 is attached |
|--------|-------|---------------------|
| 저 | formal "I/me" | 제 (제가) |
| 나 | informal "I/me" | 내 (내가) |

- ~는 attached → 저는 / 나는 (subject)
- ~를 attached → 저를 / 나를 (object)
- ~가 attached → 제가 / 내가 (subject/clause)

## How to Say "you"

Korean people rarely say "you" directly. Instead they use:
- Job titles: 부장님 (boss), 교장선생님 (principal)
- Family terms: 오빠 (older brother), 언니 (older sister)
- 아저씨/아주머니 (sir/ma'am for strangers)
- 너 (informal "you")
- 당신 ("you" — rarely used by native speakers)

## Basic Conjugation: Past, Present, Future

> **Important:** These conjugations (plain form / diary form) are rarely used in conversation. They are used in books, tests, articles, and diaries. You will learn conversational forms in Lesson 6.

### Rule for ~아/어

If the last vowel in a stem is **ㅏ or ㅗ** → add **~아**
If the last vowel is **anything else** → add **~어**
If the stem ends in **하다** → add **~여 (해)**

## Verbs

### Present Tense

| Stem ending | Rule | Example |
|-------------|------|---------|
| Consonant | Add ~는다 | 먹다 → 먹는다, 닫다 → 닫는다 |
| Vowel | Add ~ㄴ다 | 배우다 → 배운다, 가다 → 간다 |

### Past Tense

Add **~았/었다** to the stem:

| Word | Stem | Past Tense |
|------|------|-----------|
| 먹다 | 먹 | 먹었다 |
| 닫다 | 닫 | 닫았다 |
| 배우다 | 배우 | 배웠다 |
| 가다 | 가 | 갔다 |
| 오다 | 오 | 왔다 |
| 던지다 | 던지 | 던졌다 |
| 건너다 | 건너 | 건넜다 |
| 만나다 | 만나 | 만났다 |
| 공부하다 | 공부하 | 공부했다 |

**Merging rules:**
- 아 + 아 = 아 (가 + 았다 = 갔다)
- 오 + 아 = 와 (오 + 았다 = 왔다)
- 우 + 어 = 워 (배우 + 었다 = 배웠다)
- 하 + 여 = 해 (공부하다 + 였다 = 공부했다)

### Future Tense

Add **~겠다** to the stem (same for all stems):

| Word | Future |
|------|--------|
| 먹다 | 먹겠다 |
| 가다 | 가겠다 |
| 배우다 | 배우겠다 |

### Verb Conjugation Table

| Verb | Past | Present | Future |
|------|------|---------|--------|
| 먹다 | 먹었다 | 먹는다 | 먹겠다 |
| 닫다 | 닫았다 | 닫는다 | 닫겠다 |
| 배우다 | 배웠다 | 배운다 | 배우겠다 |
| 가다 | 갔다 | 간다 | 가겠다 |
| 이해하다 | 이해했다 | 이해한다 | 이해하겠다 |
| 오다 | 왔다 | 온다 | 오겠다 |
| 던지다 | 던졌다 | 던진다 | 던지겠다 |

## Adjectives

### Present Tense

**Do nothing!** Leave the adjective as-is — it is already conjugated.

- 그 선생님은 아름답다 = that teacher is beautiful
- 그 길은 길다 = that street is long
- 나의 손은 크다 = my hand is big

### Past Tense

Same rule as verbs — add ~았/었다:

| Adjective | Past Tense |
|-----------|-----------|
| 길다 | 길었다 |
| 좋다 | 좋았다 |
| 비싸다 | 비쌌다 |
| 잘생기다 | 잘생겼다 |
| 뚱뚱하다 | 뚱뚱했다 |

### Future Tense

Add **~겠다**:

| Adjective | Future |
|-----------|--------|
| 행복하다 | 행복하겠다 |
| 맛있다 | 맛있겠다 |
| 배고프다 | 배고프겠다 |

## 있다 and 있는다

**있다 is an adjective** when meaning "to have" or "to be at a location" — present tense stays as 있다.

**있다 is a verb** in certain grammatical contexts (covered in later lessons), where present tense becomes 있는다.
