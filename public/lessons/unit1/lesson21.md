# Lesson 21: Asking Questions in Korean

Source: [How to Study Korean - Unit 1, Lesson 21](https://www.howtostudykorean.com/unit1/unit-1-lessons-17-25-2/lesson-21-asking-questions-in-korean-why-when-where-and-who/)

## Question Words

| Korean | English | Example |
|--------|---------|---------|
| 뭐/무엇 | what | 이게 뭐예요? = What is this? |
| 누구 | who | 그 사람은 누구예요? = Who is that person? |
| 어디 | where | 어디에 가요? = Where are you going? |
| 언제 | when | 언제 왔어요? = When did you come? |
| 왜 | why | 왜 안 왔어요? = Why didn't you come? |
| 어떻게 | how | 어떻게 해요? = How do you do it? |
| 얼마 | how much | 얼마예요? = How much is it? |
| 몇 | how many | 몇 개 있어요? = How many are there? |

## Forming Questions

Korean questions use the same word order as statements. Add ~요 with rising intonation:

- 이거 뭐예요? = What is this?
- 어디에 가요? = Where are you going?
- 언제 와요? = When are you coming?

## Embedded Questions

When a question is inside another sentence:

- 그가 언제 오는지 알아요? = Do you know when he comes?
- 어디에 가는지 모르겠어요 = I don't know where (he/she) is going

## ~ㄹ/을 때 (When)

Used to say "when" something happens:

- 한국에 있을 때 한국어를 배웠어요 = When I was in Korea, I learned Korean
- 점심을 먹을 때 젓가락을 써요 = When eating lunch, use chopsticks
- 어렸을 때 강아지를 키우고 싶었어요 = When I was young, I wanted to raise a puppy
