# Lesson 16: Noun + ~적, ~적으로, ~적이다, ~스럽다

Source: [How to Study Korean - Unit 1, Lesson 16](https://www.howtostudykorean.com/unit1/unit-1-lessons-9-16/lesson-16/)

## ~적 (Relating to/Having properties of)

Added to Chinese-origin nouns to create descriptive words:

| Noun | +적 | Translation |
|------|-----|-------------|
| 문화 (culture) | 문화적 | cultural |
| 경제 (economy) | 경제적 | economical |
| 역사 (history) | 역사적 | historical |
| 과학 (science) | 과학적 | scientific |
| 충동 (impulse) | 충동적 | impulsive |
| 민주 (democracy) | 민주적 | democratic |
| 개인 (individual) | 개인적 | personal |

## ~적으로 (Adverb form: -ly)

| Adjective | Adverb |
|-----------|--------|
| 문화적 | 문화적으로 (culturally) |
| 경제적 | 경제적으로 (economically) |
| 역사적 | 역사적으로 (historically) |
| 과학적 | 과학적으로 (scientifically) |
| 충동적 | 충동적으로 (impulsively) |
| 민주적 | 민주적으로 (democratically) |

**Examples:**
- 한국과 미국은 역사적으로 좋은 관계가 있다 = Historically, Korea and the US have had a good relationship
- 그들은 그 문제를 과학적으로 풀었다 = They solved that problem scientifically

## ~적이다 (Adjective form)

Adding 이다 makes it an adjective that can predicate sentences:

- 미국은 민주적인 나라예요 = The US is a democratic nation
- 저 학교는 역사적인 건물이에요 = That school is a historical building
- 이것은 과학적인 문제예요 = This is a scientific problem

> ~적 is a noun; ~적이다 is an adjective. ~적이다 is more commonly used.

## ~스럽다 (Having properties of)

Added to nouns to create adjectives:

| Noun | +스럽다 | Translation |
|------|---------|-------------|
| 사랑 (love) | 사랑스럽다 | lovely |
| 자연 (nature) | 자연스럽다 | natural |
| 실망 (disappointment) | 실망스럽다 | disappointing |
| 만족 (satisfaction) | 만족스럽다 | satisfactory |
| 자랑 (pride) | 자랑스럽다 | proud |

**Examples:**
- 그 여자는 사랑스러워요 = That girl is lovely
- 그는 한국어를 자연스럽게 말해요 = He speaks Korean naturally
- 결과는 실망스러웠어요 = The results were disappointing
- 저는 우리 아들이 자랑스러워요 = I am proud of our son

**Adverb form (~게):**
- 자연스럽게 = naturally
- 사랑스럽게 = lovingly
- 실망스럽게 = disappointingly

## 실망하다 vs 실망스럽다

| Word | Type | Usage |
|------|------|-------|
| 실망하다 | verb (emotion) | 저는 실망했어요 = I was disappointed |
| 실망스럽다 | adjective (thing) | 결과는 실망스러웠어요 = The results were disappointing |

> 실망하다: Use ~에게/한테 (person) or ~에 (non-person) for "disappointed in"
> 실망스럽다: Use Subject-Object-Adjective form

## 만족하다 vs 만족스럽다

Same pattern as 실망:
- 저는 만족해요 = I am satisfied
- 결과는 만족스러웠어요 = The results were satisfactory
- 저는 결과가 만족스러웠어요 = I was satisfied with the results
