# Lesson 18: ~아/어 있다 (Being in a State)

Source: [How to Study Korean - Unit 1, Lesson 18](https://www.howtostudykorean.com/unit1/unit-1-lessons-17-25-2/lesson-18/)

## ~아/어 있다 (State of Being)

Added to verb stem to indicate being in a resulting state:

| Verb | State Form | Meaning |
|------|-----------|---------|
| 서다 | 서 있다 | to be standing |
| 앉다 | 앉아 있다 | to be sitting |
| 눕다 | 누워 있다 | to be lying down |
| 열다 | 열려 있다 | to be opened |
| 닫다 | 닫혀 있다 | to be closed |
| 잠그다 | 잠겨 있다 | to be locked |

**Examples:**
- 문이 열려 있어요 = The door is open
- 저는 의자에 앉아 있어요 = I am sitting on the chair
- 컴퓨터가 켜져 있어요 = The computer is on

## Action vs State

| Sentence | Meaning |
|----------|---------|
| 문을 열었어요 | I opened the door (action) |
| 문이 열려 있어요 | The door is open (state) |

## ~아/어 지다 (Become)

| Adjective | Become Form | Meaning |
|-----------|------------|---------|
| 크다 | 커지다 | become bigger |
| 덥다 | 더워지다 | become hotter |
| 좋다 | 좋아지다 | become better |

**Examples:**
- 날씨가 추워졌어요 = The weather got cold
- 집 값이 비싸지고 있어요 = House prices are getting expensive
