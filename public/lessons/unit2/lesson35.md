## Lesson 35: It seems to be / Might be: 것 같다

### Key Grammar Points

#### ~ㄹ/을 것 같다 — To seem like / to be likely to
Combines future tense ~ㄹ/을 것 + 같다 to express possibility or a guess.

**Future tense (guess about what will happen):**
- 비가 **올 것 같아** = It will probably rain
- 저는 친구들이랑 내일 낚시를 **할 것 같아요** = I will probably go fishing with friends tomorrow
- 새우가 너무 비쌀 **것 같아요** = The shrimp will probably be too expensive

**Present tense ~는 것 같다 (seems like, based on evidence):**
- 엄마는 기다리고 있는 **것 같아요** = It seems as though mom is waiting
- 그는 휴식을 하는 **것 같아요** = It seems like he's taking a break now

**Past tense ~ㄴ/은 것 같다 (it seems that…, based on evidence):**
- 부장님이 그 일을 이미 다 한 **것 같아요** = It seems the boss already did all the work
- 옆 집에서 사는 사람은 그 소나무를 자른 **것 같아요** = It seems the neighbor cut that pine tree

**Past tense ~았/었을 것 같다 (blind guess, no evidence):**
- 아빠가 돈을 이미 냈을 **것 같아요** = Dad probably already paid (but I haven't seen any evidence)

> **Note:** ~ㄴ/은 것 같다 implies the speaker has some evidence. ~았/었을 것 같다 is more of a blind guess.

---

#### 그렇다 + 것 같다 — Probably / I think so
- **그럴 것 같아요** = Probably / I think so / It seems as such
- **그런 것 같아요** = Probably (present tense)

| Korean | English |
|--------|---------|
| 엄마가 어디에 있어요? 병원에 갔어요? — **그럴 것 같아요** | Where's mom? Did she go to the hospital? — Probably |
| 수영장물이 다 넘쳤어요? — **그런 것 같아요** | Did all the water overflow? — Probably |

---

#### ~겠다 — Expressing Possibility
Used to express that something "must be" or "would be" based on observed evidence.

| Korean | English |
|--------|---------|
| **맛있겠다!** | That would be delicious! (looking at food) |
| **배고프겠다!** | You must be hungry! |
| **힘들겠다!** | That must be difficult! |
| **아프겠다!** | That must hurt! |
| **배부르겠다!** | You must be full! |
