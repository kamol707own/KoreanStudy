## Grammar Points

### Usage 1: When/if one action occurs, another occurs

Add **~(으)면** to the stem of a verb or adjective to create "when" or "if." The second clause is an assumption of what will happen when the first clause occurs.

**Examples:**

- 집에 도착하면 빨래를 할 거예요 = When I arrive at home, I will do laundry
- 신호를 보면 저에게 말을 바로 해 주세요 = When/if you see the signal, let me know immediately
- 과학자가 되면 피부에 습도의 영향을 연구하고 싶어요 = When/if I become a scientist, I want to study the effects of humidity on the skin
- 내일 비바람이 오면 경기가 취소될 거예요 = If it storms tomorrow, the match will be cancelled

> If the first event is certain → "when." If uncertain → "if."

**Past tense (would have):** Add **~았/었** before **~으면**:

- 내가 공부했으면 시험을 합격했을 것이다 = If I studied, I would have passed the test
- 내가 밥을 먹었으면 배고프지 않았을 것이다 = If I ate, I wouldn't have been hungry
- 햇살이 그렇게 강하지 않았으면 밖에 나갔을 거예요 = If the sunlight wasn't so strong, I would have gone outside

### Usage 2: General cause-and-effect

Indicates that when one action occurs, another generally occurs (common knowledge):

- 비가 오면 날씨가 추워져요 = When it rains, the weather gets colder
- 잠을 못 자면 다음 날에 몸이 피곤해져요 = When you don't sleep well, the next day you will be tired
- 장거리 운전을 하면 엉덩이가 아파요 = When you drive long distances, your bum will be sore
- 콜라를 매일 마시면 건강이 나빠져요 = When you drink cola every day, your health deteriorates

### Usage 3: Whenever (repeated actions)

Every time the first clause occurs, the second clause occurs:

- 피자를 먹으면 저는 콜라를 마셔요 = Whenever I eat pizza, I drink cola
- 아버지가 운전하면 습관으로 담배를 피워요 = Whenever my dad drives, he smokes cigarettes
- 내가 공부를 하면 엄마는 TV를 끈다 = Whenever I study, my mother turns off the TV

### ~ㄴ/는다면

Conjugate the verb using the plain/diary form before **~면**. This adds emphasis that the clause is a supposition:

| Tense | Example |
|-------|---------|
| Present verb | 가다 → 간다면, 먹다 → 먹는다면 |
| Present adj | 행복하다 → 행복하다면 |
| Past verb | 가다 → 갔다면, 먹다 → 먹었다면 |
| Past adj | 행복하다 → 행복했다면 |
| 이다 | 이라면 / 이었다면 |
| 아니다 | 아니라면 / 아니었다면 |

### 만약

An adverb used in sentences when the result can't be certain. Common with ~(으)면:

- 만약 내가 공부했으면 시험을 합격했을 것이다 = If I studied, I would have passed the test
- 만약 내일 비바람이 오면 경기가 취소될 거예요 = If it storms tomorrow, the match will be cancelled

### 그러면/그렇다면

By adding ~(으)면 or ~ㄴ/는다면 to 그렇다, you get "if so":

> Person 1: 내일 비가 올 것 같아요 = It will probably rain tomorrow
> Person 2: 그러면/그렇다면 공원에 안 갈 거예요 = If so, I'm not going to the park

### 아니면: "if not" / "or"

By combining 아니다 with ~면, you get "if not" / "or":

- 저는 밥을 먹고 싶어요. 아니면 죽을 것 같아요 = I want to eat rice. If not, I will probably die
- 저는 밥 아니면 사과를 먹고 싶어요 = I want to eat rice or apples
