## Grammar Points

### 1. Changing Verbs to Nouns with ~기

Add **~기** to a verb stem to turn it into a noun. Unlike ~는 것, ~기 does not describe nouns — it only creates a standalone noun.

> **가다 → 가기** (the act of going)
> **먹다 → 먹기** (the act of eating)

**Key differences from ~는 것:**
- ~는 것 can describe nouns (먹는 사람 = the person who eats)
- ~기 turns verbs into nouns but cannot describe nouns (먹기 is "eating," not "the person who eats")

### 2. ~기 시작하다 — "to start doing"

Attach ~기 before 시작하다 to express beginning an action:

- **먹기 시작하다** = to start eating
- **배우기 시작하다** = to start learning

> **어제부터 사람들이 거기서 모이기 시작했어요** = People started gathering there from yesterday

### 3. ~기 싫다 — "I don't want to do"

Express reluctance by attaching ~기 before 싫다:

- **먹기 싫어** = I don't want to eat
- **가기 싫어** = I don't want to go

### 4. Common ~기 Noun Forms

Some nouns are commonly formed with ~기:

| Verb | ~기 Noun | Meaning |
|------|----------|---------|
| 달리다 | **달리기** | running (jog) |
| 넘다 | **줄넘기** | jump rope |
| 쓰다 | **쓰기** | writing (test section) |
| 읽다 | **읽기** | reading (test section) |
| 듣다 | **듣기** | listening (test section) |

### 5. Making Lists with ~기

Use ~기 at the end of items in to-do lists:

- **커피를 만들기** = Make coffee
- **책상 정리하기** = Organize desk
- **책을 매일 읽기** = Read books every day

### 6. Computer Buttons with ~기

Korean UI buttons often use ~기 noun forms:

| Verb | Button | Meaning |
|------|--------|---------|
| 보다 | **보기** | View |
| 열다 | **열기** | Open |
| 닫다 | **닫기** | Close |
| 찾다 | **찾기** | Find/Search |
| 보내다 | **보내기** | Send |

### 7. Changing Verbs/Adjectives to Nouns with ㅁ/음

Add **ㅁ** to stems ending in a vowel, **음** to stems ending in a consonant:

| Word | + ㅁ/음 | Meaning |
|------|---------|---------|
| 싸우다 | **싸움** | a fight |
| 죽다 | **죽음** | death |
| 도다 | **도움** | help |
| 걷다 | **걸음** | step |
| 느끼다 | **느낌** | feeling |
| 살다 | **삶** | life (ㄹ irregular) |
| 그렇다 | **그럼** | of course (ㅎ irregular) |

> **Note:** For 하다 verbs, simply remove 하다 (설명하다 → 설명).

### 8. 그럼 — "Of course / Yes"

그렇다 + ㅁ/음 = **그럼**. Used to affirm a question:

- **운동을 매일 해요?** → **그럼요** = Do you exercise every day? → Of course

## Example Sentences

- **나는 형이랑 싸움에서 이겼어** = I won in a fight with my brother
- **나는 아빠의 죽음을 잊지 않았어** = I didn't forget the death of my father
- **나는 밥을 벌써 먹기 시작했어** = I already started eating
- **쌀을 씻기 싫어요** = I don't want to wash the rice
- **시험은 어땠어? 쓰기랑 듣기는 너무 어려웠어** = How was the exam? Writing and listening were really hard
- **제가 강아지를 기르고 싶은지 모르겠어요** = I don't know if I want to raise a puppy
- **사람들이 모두 볼 수 있게 안내문을 벽에 붙였다** = I posted a sign on the wall so everyone could see
