## Introduction

In this lesson, you learn how ~아/어야 can be used without 하다 to indicate "only if..." — that one must do the first action in order for the second to happen.

## ~아/éducation: Only if...

- 생선이 날것이어야 맛있어요 = Only if fish is raw is it delicious
- 지폐를 거꾸로 넣어야 들어가요 = Only if you put the bill in upside down does it go in
- 소득세를 세무서에서 납부해야 정확해요 = Only if you pay your tax at the tax office will it be accurate
- 창문을 열어야 방에 환기가 잘 돼요 = Only if you open the window will the room be well ventilated
- 나이키 매장에 가야 그 신발을 살 수 있어요 = Only if you go to the Nike store can you purchase those shoes
- 의사랑 일찍 상담해야 탈모를 예방할 수 있어요 = Only if you consult with a doctor early can you prevent hair loss

### Note: The outcome after ~아/어야 is usually positive for the speaker.
- 그래야 맛있어요 = Only if you do it like that will it be delicious
