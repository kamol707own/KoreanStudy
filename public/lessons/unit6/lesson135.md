## Introduction

This lesson discusses three meanings of the syllable 성: 性 (character/range of quality), 成 (completion), and 誠 (loyalty/effort/devotion).

## 성 (性) = Character or a Range of Some Quality

When added to -하다 adjectives, the newly created noun refers to the range or degree:
- 가능하다 → **가능성** = possibility
- 중요하다 → **중요성** = importance
- 진실하다 → **진실성** = truthfulness
- 위험하다 → **위험성** = dangerousness
- 독창하다 → **독창성** = creativity
- 필요하다 → **필요성** = necessity
- 다양하다 → **다양성** = diversity
- 전염하다 → **전염성** = contagiousness

### Without ~하다 form
- 산성 = acidity, 지성 = intelligence, 적성 = aptitude
- 관성 = inertia, 감성 = sensitivity, 독성 = toxicity
- 효율성 = effectiveness, 사회성 = sociability
- 참을성 = patience

## 성 (成) = Completion
- 성공 = success, 완성 = completion, 형성 = development
- 성과 = result, 구성 = formation, 조성 = creation

## 성 (誠) = Loyalty, Effort, Devotion
- 정성 = effort/devotion, 충성 = loyalty
- 성의 = sincerity, 성실하다 = to be faithful
