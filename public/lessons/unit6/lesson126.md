## Introduction

In this lesson, you will learn how to add ~별 to nouns and the meaning it creates. You will see that it is often attached to nouns to describe another noun, or used with ~로 to act as an adverb.

## To be Grouped or Classified by: ~별(로)

~별 is added to nouns that represent more than one type or level of something. When this is done, it classifies the different types/levels and indicates that something is done (separately) for each type/level.

Examples:
- 수준 = level → **수준별**: Something will be done separately for each level
- 성 = sex/gender → **성별**: The sexes will be classified
- 학년 = a grade level in school → **학년별**: Grades will be classified
- 부서 = department → **부서별**: Departments will be classified
- 개인 = personal, individual → **개인별**: Individuals will be classified
- 국가 = nation → **국가별**: Nations will be classified

These constructions are often placed before other nouns to describe them:
- **수준별 수업** = a class divided by levels
- **성별 임금 차이** = the difference in wage between the two sexes
- **학년별 명단** = a list of students names based on grade
- **부서별 회의** = a meeting for each department
- **개인별 평가** = an evaluation done for each individual
- **국가별 언어 설정** = the language setting for each nation

### Example Sentences
- 수준별 수업은 좋은 점도 있고 나쁜 점도 있어요 = Classes divided by levels have good points and bad points, too
- 대부분 회사에는 성별 임금 차이가 있어요 = At most companies, there is a wage difference between sexes
- 개학 전에 우리는 학년별 명단을 작성해야 돼요 = Before the start of school, we need to make a list of students names by grade
- 연말이라서 부서별 회식이 많아요 = It is the end of the year, so there are a lot of staff dinners among departments
- 새로운 컴퓨터를 사면 국가별 언어 설정을 해야 돼요 = When you buy a new computer, you need to set the language setting

### Using ~별(로)

~별로 is commonly used to indicate that the action will be different depending on the types/levels specified.

- 우리 회사는 분기별로 세금을 내요 = Our company pays tax by the quarter
- 코치가 선수들을 체중별로 나눴어요 = The coach divided the athletes by weight
- 저는 신분증을 나이별로 다 정리했어요 = I organized the ID cards by age
- 모든 나라가 국가별로 대표를 지명했어요 = Each country designated a representative separately
- 대륙별로 발견된 공룡들의 무게는 달랐어요 = The weight of dinosaurs was different depending on the continent
- 사람의 평균 월급은 보통 계급별로 다릅니다 = The average pay of a person is different depending on class
- 한국에서는 영어와 수학수업이 보통 수준별로 나눠져요 = In Korea, English and Math classes are usually divided by level
