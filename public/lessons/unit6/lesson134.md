# Lesson 134: Ultimate Konglish Guide

## Introduction

Korean has borrowed many words from English. While many are directly recognizable (마우스, 키보드, 컴퓨터), many "Konglish" words have different meanings in Korean than their English counterparts.

## Words Borrowed from English with Different Meanings

- **아파트** = (apart) = apartment building (typically taller, rectangular buildings)
- **오피스텔** = (office-tel) = regular western style apartment
- **원룸/투룸/쓰리룸** = (one/two/three room) = studio/two-room/three-room apartment
- **서비스** = (service) = complimentary things (usually food) when purchasing something
- **헬스** = (health) = weight training at a fitness center
- **스킨십** = (skinship) = public displays of affection
- **PPT** = (PowerPoint) = PowerPoint slideshow
- **프린트** = (print) = notes handed out at a school
- **햄** = (ham) = spam
- **디스크** = (disc) = herniated disc in one's back
- **컨디션** = (condition) = used when telling people that one is sick
- **린스** = (rinse) = conditioner
- **MT** = (Membership Training) = going on a retreat with fellow students
- **CC** = (Campus Couple) = two people dating at the same university
- **팩** = (pack) = face mask for skin care
- **실버타운** = (silvertown) = retirement complex for seniors
- **셀카** = (self-camera) = taking a selfie
- **미팅** = (meeting) = a group date / speed dating
- **사이다** = (cider) = Sprite or 7-Up tasting beverage
- **펜션** = (pension) = cottage in the countryside that people rent
- **내비게이션** = (navigation) = a GPS system for driving
- **노트북** = (notebook) = laptop
- **런닝머신** = (running machine) = treadmill
- **렌즈** = (lens) = contact lenses
- **파이팅/화이팅** = (fighting) = good luck!
- **풀옵션** = (full option) = furnished apartment
- **패딩** = (padding) = thick, padded winter jacket
- **개그** = (gag) = comedy
- **코팅** = (coating) = laminate
- **에이드** = (ade) = carbonated fruit drinks
- **치킨** = (chicken) = fried chicken
- **컨닝** = (cunning) = cheating on an exam
- **오픈카** = (open car) = convertible
- **멀티룸** = (multi-room) = a room with many entertainment items for kids

## Words that are a Combination of English and Korean

- **헬기** = hel(icopter) + 기 (machine) = helicopter
- **믹서기** = mixer + 기 (machine) = blender
- **드럼세탁기** = drum + 세탁기 (washing machine) = front-loading washing machine
- **골프장** = golf + 장 (place) = golf course
- **트레이닝복** = training + 복 (clothing) = training clothes
- **셀카봉** = selfie + 봡 (stick) = selfie stick
- **싱크대** = sink + 대 (desk) = sink
- **노잼** = no + 재미 (fun) = not funny
- **팀장** = team + 장 (boss/leader) = team leader
- **왕팬** = 왕 (king) + fan = a big fan

## Direct Borrowings from English

Korean uses hundreds of words directly borrowed from English with the same meaning, including: 마우스, 키보드, 컴퓨터, 스크린, 폰, 박스, 초콜릿, 시리얼, 게임, 골프, 데이터, 드라마, 뉴트워크, 라디오, 로봇, 마사지, 마스크, 버튼, 소다, 쇼핑, 스타, 스파게티, 스포츠, 시스템, 에어컨, 이메일, 커피, 케이크, 티켓, 파티, 프로, 프로그램, 필터, 핑크, and many more.
