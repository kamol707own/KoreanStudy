## If one intends to: ~(으)려면

~(으)려고 (Lesson 32) + ~(으)면 (Lesson 43) = ~(으)려면

Meaning: "if one intends to" or "if one wants to"

Sentences often end with ~아/어야 하다, ~이/가 필요하다, or imperative endings.

### Examples

- **그렇게 비싼 것을 사려면 돈이 많이 필요해요** = If you intend to buy something expensive, you need a lot of money
- **서울에 가려면 지하철을 타세요!** = If you intend to go to Seoul, take the subway!
- **이 일을 끝내려면 인내심이 많이 필요해요** = If you intend to complete this task, you need a lot of patience
- **뒷맛을 없애려면 새콤한 것을 먹어야 돼요** = If you want to get rid of the aftertaste, eat something sour
- **예쁜 경치를 보려면 아침 일찍 일어나야 돼요** = If you want to see the beautiful scenery, wake up early
- **자기개발에 힘쓰려면 돈을 낭비하지 말고 매달 저축하는 게 중요해요** = If you want self-improvement, don't waste money and save monthly
