## Would: ~ㄹ/을 텐데(요)

텐데 is created by merging the pseudo-noun 터 + 이다 + ~ㄴ/은데. 터 provides connotation of expectation or guess.

### If... (past tense, with regret)

~ㄹ/을 텐데 adds a feeling of **regret** when used with hypothetical past tense:

- **날씨가 좋았더라면 저는 공원에 갔을 텐데요** = If it were nice out, I would have gone (but it wasn't)
- **제가 돈이 있었더라면 그것을 샀을 텐데요** = If I had money, I would have bought that

### Even if... (annoyance)

~ㄹ/을 텐데 adds a feeling of **annoyance** with ~더라도:

- **날씨가 좋았더라도 저는 공원에 안 갔을 텐데요** = Ugh, even if it were nice, I still wouldn't have gone

### If... (present tense, "too bad")

- **음식이 더 있으면 좋을 텐데** = Oh, it's too bad there isn't more food

### More Examples

- **지금 안 가면 그 품목이 없을 텐데** = If we don't go now there won't be any items
- **주사를 안 맞았더라면 팔이 가렵지 않았을 텐데** = If I didn't get a vaccination, my arm wouldn't be itchy
- **큰길로 갔더라면 모임에 늦게 도착하지 않았을 텐데** = If we went via the main road, we wouldn't have arrived late

## It is probable that: ~ㄹ/을 텐데(요)

Used when the speaker supposes or expects something. Slightly more sure than ~ㄹ/을 것 같다, with a slight feeling of worry.

- **비가 올 텐데** = It will probably rain (I'm a bit worried)
- **날씨가 추울 텐데** = The weather will probably be cold
- **그 식당이 이미 닫았을 텐데** = The restaurant is probably already closed (that's too bad)

---

## In the middle of a sentence

~ㄹ/을 텐데 connects two clauses meaning "it is probable that... so":

- **길이 막힐 텐데 지하철로 가자** = The roads will probably be jammed, so let's take the subway

~ㄹ/을 테니까 is similar and more natural when expressing **intention**:

- **지금 갈 테니까 조금 더 기다려 주세요** = I'm going to leave now, so wait just a little bit longer
- **제가 돈을 많이 벌 테니까 걱정하지 마세요** = I am going to earn a lot of money, so don't worry
